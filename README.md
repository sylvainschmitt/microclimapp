## Installation

You can install the latest version of **miroclimapp** from Github using the [`devtools`](https://github.com/r-lib/devtools) package:

``` r
if (!requireNamespace("devtools", quietly = TRUE))
  install.packages("devtools")

devtools::install_github("sylvainschmitt/miroclimapp")
```

## Usage

```r
library(miroclimapp)
run_app()
```
